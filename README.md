# MISIRI Core

MISIRI es un núcleo de CPU de hardware abierto.
Su nombre es inspirado en la lengua y la tradición metalúrgica purépecha.

El nombre *misiri* proviene del idioma purépecha (Pʼurhépecha) y significa
**“chispa”**, símbolo de energía, activación y el inicio del proceso de
cómputo.

MISIRI Core está pensado para ser el corazón de procesamiento del MCU **TANIK**,
enfocado en control embebido, procesamiento de señales e inteligencia en el
borde (edge computing), con aplicaciones reales en contextos industriales y
productivos.

## Características principales
- Compatible con RISC-V (base RV32I)
- Microarquitectura simple y determinista
- Soporte para extensiones de instrucciones personalizadas
- Diseño orientado a FPGA y factible para ASIC de bajo costo
- Enfoque en uso práctico y aplicaciones reales

## Filosofía de diseño
MISIRI sigue una filosofía de “chispa”:

Un núcleo pequeño, eficiente y preciso, cuyo objetivo no es hacerlo todo,
sino **activar y coordinar hardware especializado** de forma determinista y
eficiente.
